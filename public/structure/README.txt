PANDUAN MENGGANTI FOTO STRUKTUR ORGANISASI
=========================================

Agar foto muncul di website, silakan letakkan file foto (format JPG/JPEG/PNG) di folder ini (`public/structure/`) 
dengan nama file yang SESUAI (PERHATIKAN HURUF KECILNYA):

1. Wali Kelas    ->  wali-kelas.jpg
2. Ketua Kelas   ->  ketua.jpg
3. Wakil Ketua   ->  wakil.jpg
4. Bendahara 1   ->  bendahara-1.jpg
5. Bendahara 2   ->  bendahara-2.jpg
6. Sekretaris 1  ->  sekretaris-1.jpg
7. Sekretaris 2  ->  sekretaris-2.jpg

CATATAN:
- Pastikan ekstensi file adalah .jpg (jika .png atau .jpeg, silakan rename atau convert).
- Untuk hasil terbaik, gunakan foto dengan rasio 1:1 (Persegi).
